TechNova - Tienda Tecnológica y de ropa.

Proyecto final: una tienda online que consume una API de productos, permite agregar al carrito, modificar cantidades, eliminar items y simular una compra.

Funcionalidades:

*Mostrar productos desde FakeStore API.

*Agregar productos al carrito.

*Actualizar cantidades en tiempo real.

*Eliminar productos.

*Guardar carrito en LocalStorage.

*Calcular total automáticamente.

*Botón Pagar que simula una compra exitosa.

*Formulario de contacto funcional mediante Formspree.

Tecnologías utilizadas:

-HTML5

-CSS3

-JavaScript

-API pública: https://fakestoreapi.com/

-Formspree para el envío de formularios.
